To the access the data used in this project, click the following link: https://archive.ics.uci.edu/dataset/908/realwaste. Then, click the "Download" button on the right side of the page. The data will download as a zip file called "RealWaste", which contains the folders "Vegetation", "Textile Trash", "Plastic", "Paper", "Miscellaneous Trash", "Metal", "Glass", "Food Organics", and "Cardboard". Each folder then contains various images of each type of trash defined by the folder name.
