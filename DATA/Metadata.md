# Summary of Data

The dataset is contained in a zip file titled **“realwaste.”** This file includes **9 subfolders**, each representing a different category of waste:

- Cardboard  
- Food Organics  
- Glass  
- Metal  
- Miscellaneous Trash  
- Paper  
- Plastic  
- Textile Trash  
- Vegetation  

Each subfolder contains multiple `.jpeg` images corresponding to its waste category.  
In total, the dataset is **672,408 KB** in size and contains **4,752 images**. The data is currently stored individually on each team member’s computer.

---

# Provenance

The data originated from **Sam Single, Saeid Iranmanesh, and Raad Raad**, who photographed waste items from a landfill.  
The dataset was posted to the **UCI Machine Learning Repository** on **October 17, 2023**, and can be accessed here:  
<https://archive.ics.uci.edu/dataset/908/realwaste>.

Each team member downloaded the data on **November 10, 2025**. Since then, the dataset has been modified for project use. All modifications and results are documented in the team’s GitHub repository:  
<https://github.com/jessiebailey/Project-3.git>.
